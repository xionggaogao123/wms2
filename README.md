# 仓库管理系统


## 环境准备



```bash
JDK 1.8; 
Redis
Mysql >= 5.7
```

## 提供的脚本

### Start project

```bash
Application main start
```

### Build project

```bash
maven package
```

### Generator code

```bash
CodeGenerator main start
```
